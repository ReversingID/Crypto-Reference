## Lightweight Encryption Algorithm (LEA) Cipher

* Pencipta:
    - Deukjo Hong
    - Jung-Keun Lee
    - Dong-Chan Kim
    - Daesung Kwon
    - Kwon Ho Ryu
    - Dong-Geon Lee
* Kategori: Block Cipher
* Ukuran block (bit): 128
* Ukuran kunci (bit): 128 / 192 / 256
* Jumlah round: 24 / 28 / 32
* Struktur: ARX (modular Addition, bitwise rotation, bitwise XOR)
